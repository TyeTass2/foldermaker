# Windows 10+ Folder Generator

**Why pay for folder management software when you can automate your workflow for free?**

Originally developed for 3D printing projects, this lightweight utility is perfect for anyone looking to organize their digital workspace without the bloat. 

Because this is a transparent PowerShell script, it is as safe as software gets. Unlike many commercial tools that track user data, this script does one thing: it creates folders. It is easy to audit, simple to modify, and yours to customize however you please.

---

### Features
* **Zero Configuration:** One-click install with no manual PowerShell setup required.
* **Smart Paths:** Save projects directly to your Desktop or a custom directory.
* **Flexible Naming:** Support for Numerical, Prefix, and Suffix batching.
* **Ready for Workflow:** The launcher can be pinned to your Taskbar for instant access.

### Installation
1. Download `Install_FolderGenerator.bat` from this repository.
2. Double-click `Install_FolderGenerator.bat` to begin the automated setup.
3. Launch the utility via the **Project Folder Generator** icon now on your Desktop.
